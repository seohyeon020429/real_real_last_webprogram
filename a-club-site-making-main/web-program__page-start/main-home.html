코드에 쓴 태그에 대해 알려줘!!

nav,div,a 태그에 대해 설명해줄게!!

nav 요소는 위 사진에서 사이트 이름, 사이트 소개, 동아리, 로그인 버튼을 누르면

해당 사이트로 가는 링크를 보여주는 구획을 나타내!!


div 태그는 순수한 컨테이너로서 본질적으로 아무것도 나타내지 않아.

하지만 컨텐츠의 구역을 나눌 때 사용해!! 한 눈에 코드를 파악하기 쉽거든!!


a 태그는 링크야!! nav가 링크를 보여주는 구획이라면

a 는 href 를 통해 링크를 만들어!!

직접 실습해볼래!!

우선 CSS를 적용한 코드를 보여줄게~!


          <nav class = "nav_bar">
          <div class = "top_nav">
              <a class = "site-name" href="index.html">사이트 이름</a>
              <a class = "center-link" href="clubs_introduce.html">사이트 소개</a>
              <a class = "center-link" href="clubs.html">동아리</a>
              <a class = "center-link" href="apps.html">대학생활 필수 앱</a>
              <a class = "top_nav-login" href="login.html">로그인</a>
          </div>
          </nav>

top-nav 클래스 소개

site-name이랑 center-link 클래스 소개

top_nav-login 클래스 소개
